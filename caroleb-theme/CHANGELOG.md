# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial theme release
- WordPress classic theme structure
- SCSS 7-1 architecture
- WordPress Coding Standards configuration (phpcs.xml)
- ESLint configuration for JavaScript
- Stylelint configuration for SCSS
- EditorConfig for consistent formatting
- NPM scripts for building and linting
- Composer configuration for PHP dependencies
- ACF integration with JSON sync
- Template files (header, footer, sidebar, index, single, page, archive, search, 404)
- Template parts for content display
- Helper functions (template-tags.php, template-functions.php)
- Navigation menu support (primary and footer)
- Widget areas (sidebar and footer)
- Post thumbnail support
- Custom logo support
- HTML5 markup support
- Responsive embeds support
- Translation ready
- Comprehensive README documentation
- Comments template
- Search form template

### Features
- Mobile-responsive layout
- Clean, educational code structure
- Well-commented code for learning
- Modern WordPress theme development practices
- Accessible markup with ARIA labels
- Skip link for accessibility
- Screen reader text utilities

### Development Tools
- Sass compilation with watch mode
- Auto-formatting scripts
- Code quality linting
- Build scripts for production
